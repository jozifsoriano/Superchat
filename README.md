#Compilation Instructions:
0. Fix makefile to work with your asio directory.
1. Initial make
```
make
```
2. Run server in one terminal.
```
make runs
```
3. Run client as many times as necessary to open chat windows.
```
make runc
```

4. To clean, run on terminal:
```
make clean
```
